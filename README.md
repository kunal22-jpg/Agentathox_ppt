# NUTRACIA — AgentathonX 2026 Presentation

A browser-based slide deck deploying all 14 NUTRACIA presentation slides. No frameworks, no build step — pure HTML.

## 📁 Structure

```
nutracia-ppt/
├── index.html          ← main presenter UI
├── vercel.json         ← Vercel config
└── slides/

    ├── s2.html         ← Problem Statement

```

## 🚀 Deploy to Vercel

### Option 1 — Vercel CLI
```bash
npm i -g vercel
cd nutracia-ppt
vercel
```

### Option 2 — Vercel Dashboard (recommended)
1. Push this folder to a GitHub repo
2. Go to https://vercel.com/new
3. Import the repo → click **Deploy**
4. Your URL: `https://nutracia-ppt.vercel.app` (or your custom domain)

## ⌨️ Keyboard Controls

| Key | Action |
|-----|--------|
| `→` / `Space` | Next slide |
| `←` | Previous slide |
| `F` | Fullscreen |
| `T` | Toggle thumbnail panel |
| `Home` / `End` | First / Last slide |
| `1–9` | Jump to slide |
